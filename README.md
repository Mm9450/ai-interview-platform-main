# Prepped – AI Mock Interview Marketplace

Prepped is a full-stack AI-powered mock interview platform that connects interviewees with interviewers for live interview sessions. The platform supports real-time video calls, AI-generated interview questions, automated feedback reports, and a credit-based booking system.

Built using Next.js App Router, Supabase PostgreSQL, Clerk authentication, Stream SDK, Tailwind CSS, and Shadcn UI, the application provides a scalable and modern SaaS-style architecture with responsive UI and secure role-based workflows.

## Features

- Dual-user workflow for Interviewers and Interviewees
- Secure authentication and onboarding with Clerk
- Real-time video calling, screen sharing, and chat using Stream SDK
- AI-powered interview question generation
- Automated AI feedback and performance reports
- Credit-based booking and monetization system
- Interview scheduling and availability management
- Responsive and reusable UI components
- Protected routes and session handling
- Modern dashboard interface for managing interviews and earnings

## Tech Stack

- Next.js (App Router)
- React.js
- Supabase PostgreSQL
- Clerk Authentication
- Stream Video SDK
- Tailwind CSS
- Shadcn UI

## Purpose

This project was built to simulate a production-grade interview marketplace platform while learning scalable frontend architecture, real-time communication systems, authentication workflows, and AI-integrated user experiences.
